MqttKafkaBridge documentation
===========================================

The MQTT Kafka  Bridge is a java application that serves to create a connection between MQTT and a Kafka cluster so messages can be passed between them. This particular application is built from Springboot, can be contenarized and can also be controlled using some endpoints.
